FastAPI Endpoint

This commission engine is available via REST API for integration with your systems:

POST /api/commissions/calculate
Headers:
  Content-Type: multipart/form-data
  Authorization: Bearer <token>

Body:
  file: payroll.csv
  config: {
    "base_rate": 7,
    "tiers": [...],
    "period": "quarterly"
  }
Response: JSON with commission calculations, accruals, and reconciliation data
